# Introduction

Welcome to **DocCraft PDF**, a powerful microservice for converting Markdown documents into beautiful, professionally styled PDFs.

## What is DocCraft PDF?

DocCraft PDF is a Node.js-based microservice that leverages the power of Pandoc and XeLaTeX to transform your Markdown content into polished PDF documents with custom branding, headers, footers, and styling.

## Key Features

- **Multiple File Support**: Combine multiple Markdown files into a single PDF
- **Custom Branding**: Add your logo, custom headers, and footers
- **Metadata Control**: Set title, subtitle, author, and date
- **Table of Contents**: Automatic TOC generation with configurable depth
- **Syntax Highlighting**: Beautiful code blocks with syntax highlighting
- **Smart Caching**: Avoid regenerating identical PDFs
- **RESTful API**: Simple HTTP endpoint for PDF generation
- **CLI Tool**: Generate PDFs from the command line

## Use Cases

DocCraft PDF is perfect for:

- **Technical Documentation**: Convert your project docs into distributable PDFs
- **Reports**: Generate branded reports from Markdown content
- **Ebooks**: Create professional ebooks from your writing
- **Presentations**: Transform your notes into presentation handouts
- **Contracts & Proposals**: Professional business documents

## Architecture

The microservice is built with:

- **Node.js + Express**: Fast, lightweight HTTP server
- **Pandoc**: Universal document converter
- **XeLaTeX**: High-quality PDF typesetting
- **YAML**: Configuration-driven styling
